# PDF Zusammenfügen - pdfzus

Ich habe ein neues Community-Plugin für Obsidian veröffentlicht: **PDF Zusammenfügen - pdfzus**.

Das Plugin löst einen sehr einfachen, aber im Alltag häufigen Fall:

- mehrere PDF-Dateien auswählen
- Reihenfolge direkt in Obsidian prüfen und anpassen
- Ergebnis lokal speichern

Wichtig war mir dabei vor allem Datenschutz und ein klarer Workflow:

- keine Uploads
- kein Konto
- keine Telemetrie
- keine Verarbeitung auf einem externen Server

Das Plugin ist bewusst auf **Obsidian Desktop** fokussiert, weil es mit lokalen Dateien arbeitet und den nativen Dateidialog nutzt, wenn verfügbar.

## Kurz gesagt

Wenn Sie PDFs regelmäßig lokal bündeln müssen und dafür keinen Browserdienst verwenden möchten, ist das Plugin für genau diesen Anwendungsfall gebaut.
